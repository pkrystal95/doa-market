# DOA Market User App - 테스트 시나리오

## 개요
이 문서는 DOA Market User App의 포괄적인 테스트 시나리오를 정의합니다.

## 1. 인증 (Authentication) 테스트

### 1.1 회원가입
**우선순위**: HIGH

| 시나리오 ID | 설명 | 입력 | 예상 결과 | 상태 |
|------------|------|------|----------|------|
| AUTH-REG-001 | 정상적인 회원가입 | 유효한 이메일, 비밀번호, 이름 | 회원가입 성공, 자동 로그인 | 구현 필요 |

| AUTH-REG-003 | 잘못된 이메일 형식 | 잘못된 이메일 | 유효성 검사 실패 | 구현 필요 |
